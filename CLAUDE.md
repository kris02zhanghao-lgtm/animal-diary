# 动物偶遇图鉴

## 项目描述
用 AI 记录城市中偶遇小动物的个人图鉴 web app。用户上传照片，AI 自动识别动物种类并生成趣味日志，保存到本地列表查看。

## 技术栈
- React（Vite）
- Tailwind CSS
- OpenRouter API（封装为 aiService.js 模块）
- localStorage
- Vercel 部署
