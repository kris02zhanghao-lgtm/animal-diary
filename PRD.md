# 动物偶遇图鉴 · 产品需求文档
> 版本：v1.0 规划中 | 更新日期：2026.05.04

---

## 一句话描述

一款用AI记录你在城市中偶遇小动物的个人图鉴app，拍照即可自动识别物种、生成趣味日志，越用越丰富。

---

## 要解决的问题

路上碰到可爱的小动物会随手拍，但这些照片散落在相册里，没有专门的地方记录、回顾和赋予意义。现有工具要么太学术（iNaturalist），要么太通用（相册/备忘录），缺少一个轻量、有趣、有情感温度的个人动物偶遇记录产品。

---

## 目标用户

喜欢随手拍小动物的城市年轻人（不是专业观鸟/自然爱好者，就是普通的"看到猫猫狗狗会停下来拍照"的人）。

---

## 与竞品的核心差异

| 维度 | iNaturalist/Seek | 校园猫谱 | 本产品 |
|------|-----------------|---------|--------|
| 定位 | 公民科学工具 | 校园流浪猫管理 | 个人情感记录 |
| 用户 | 自然研究者 | 校园猫友社群 | 普通城市年轻人 |
| 识别目标 | 精确物种分类 | 已知校园猫个体 | 生活化物种识别 |
| 核心体验 | 科研数据贡献 | 社区信息共享 | 趣味日志+地图收集 |
| AI能力 | 物种分类 | 拍照识猫 | 物种识别+趣味文案生成 |

## 功能规划

### v0.1-v0.3：基础记录 & 后端上线 ✅ 完成

- **照片上传 + AI识别 + 日志生成**：拍照识别、编辑、保存
- **偶遇列表**：时间线视图、季度分组、卡片展开详情
- **删除确认**：二次确认弹窗防止误删
- **后端迁移**：从localStorage迁到Supabase，Vercel部署上线

---

### v0.4-v0.7：浏览维度 & 内容分享 ✅ 完成

- **偶遇地图**：地理位置标记、高德地图集成、自动定位
- **地点编辑**：LocationPicker组件、POI搜索、地图反向编码
- **图鉴收集**：按物种大类聚合、多层网格浏览、完成度统计
- **偶遇报告**：年度/季度总结、核心数据展示、AI洞察文案
- **分享卡片**：单条记录分享、Canvas生成分享图片

---

### v0.8-v0.9.1：智能识别 & 游戏化 ✅ 完成

- **回头客识别**：同物种相似度判断、自动弹窗确认、双向关联展示
- **AI个性化总结**：时间窗口内的行为画像、物种/地点聚合洞察
- **成就系统**：5个成就规则、解锁通知、报告页成就展示
- **物种修正交互**：大类二级选择、"其他"自填、自动分类兜底
- **分享图鉴**：按大类生成分享卡片、公开访问链接、图鉴二维码

---

### v1.0：数据驱动 & 体验优化 🚀 待开始

- **数据埋点**：识别成功率、保存转化率、分享率、成就解锁率等关键指标
- **AI模型评估**：对标Gemini vs Claude vs GPT-4的准确率/延迟/成本
- **识别失败兜底**：超时/失败时提供空卡片、允许用户手填物种和日志
- **国内域名支持**：替换vercel.app为国内可访问域名

---

## MVP成功标准

- [x] 能上传一张照片，AI能识别出动物种类

## 技术方案

| 项目 | 方案 | 备注 |
|------|------|------|
| 框架 | React（Vite） | |
| 样式 | Tailwind CSS | |
| 数据存储 | Supabase PostgreSQL + Auth | 用户认证（匿名自动登录）+ 行级安全（RLS） |
| 后端 | Vercel Functions | 代理 OpenRouter API，隐藏 API Key |
| AI调用 | OpenRouter API（后端代理） | 默认模型：google/gemini-2.5-flash-lite |
| 图片处理 | FileReader → base64压缩 | |
| 部署 | Vercel | 免费，支持跨设备数据同步 |

> ✅ 安全性：OpenRouter API Key 存储在 Vercel 后端环境变量，不暴露到前端。Supabase RLS 确保用户数据隔离。可安心分享给他人使用。

## 风险与应对

| 风险 | 应对 |
|------|------|
| AI物种识别不够准 | 允许用户手动修正；识别粒度放粗（"橘猫"而非精确亚种） |
| 日志生成质量不稳定 | 精调prompt；允许用户编辑 |
| 使用频次低 | 成就系统制造主动寻找动机；图鉴完成度激发收集欲 |
| 回头客识别不准 | 定位为"可能是"而非"确定是"，用户手动确认 |

## 一句话总结项目价值（面试用）

这个项目展示了完整的AI产品能力：多模态识别（视觉理解）、LLM内容生成（文案创作）、地理信息系统（地图可视化）、游戏化设计（成就系统），同时体现了从真实个人需求出发的产品洞察力。
